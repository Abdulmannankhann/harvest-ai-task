# Frontend Task Assignment for Harvest AI

Hi! Welcome to the frontend task assignment for Harvest AI. This project is built using React, TypeScript, and JavaScript. The main dependencies utilized in this project include Axios for API calls and Material UI for design and components.

If you're new to this project, feel free to explore and familiarize yourself with the codebase. Below, you'll find some key information to get you started:

## Project Overview

- **Tech Stack:**

  - React
  - TypeScript
  - JavaScript

- **Dependencies:**
  - Axios for API calls
  - Material UI for design and components

## Instructions

1. **Clone this repository to your local machine.**

   - Use the command: `git clone https://github.com/Abdulmannankhann/harvest-ai-task.git`

2. **Install the necessary dependencies using your package manager.**

   - Navigate to the project directory and run: `npm install`

3. **Explore the project structure and dive into the codebase.**

   - Familiarize yourself with the directory structure and key files.

4. **Run the code using npm start.**

   - Execute: `npm start`
   - Open your browser and visit `http://localhost:3000` to view the application.

5. **Complete the assigned tasks outlined in the frontend task.**

   - Refer to the task document for specific instructions.

6. **See the demo of the application.** - Click [here](https://abdul-harvest-ai-task.netlify.app/) to access the live demo.

Feel free to reach out if you have any questions or need further assistance. Happy coding!
